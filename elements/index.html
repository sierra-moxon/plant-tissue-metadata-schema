    Plant Tissue Metadata Schema

    A LinkML schema for capturing metadata about plant tissue samples, including experimental conditions, treatments, and sample processing information. Designed for compatibility with MIXS standards.

    URI: https://w3id.org/plant-tissue-metadata-schema

    Name: plant-tissue-metadata-schema

    Classes

    Class Description
    PlantTissueSample A sample of plant tissue with associated metadata
